## 📌 About

This repository contains my personal implementations of classic **Gang of Four (GoF)** design patterns in Python.

The purpose of this project is to:

- Strengthen architectural thinking
- Practice clean and maintainable design
- Explore Pythonic adaptations of classic patterns

This is a learning-oriented repository — not a production library.

## 📦 Implemented Patterns

### Creational Patterns

- Factory (Simple Factory / Factory Method)
- Singleton *(DependencyManager example with a typed registry)*
